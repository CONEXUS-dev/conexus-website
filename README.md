## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🚀 Deployment

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Deploy to production
npm run build
npm run start

# Deploy to Vercel
npx vercel --prod --yes
```
